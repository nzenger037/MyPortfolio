Video Game: Hyper Flow

Game Dev Project

Hyper flow is a retro-wave hover ship racing game developed in the unity engine, utilizing real physics to thrust the ship into a hover state causing any weight or balance shifts realistically affect how the ship handles.

You can race against another hover racer AI using boost power ups, ramps, and projectile laser attacks to best your enemy and take the gold.

This was one of my first solo game dev projects which taught me a lot about game engine physics and how to manipulate them with C# scripts, working on adding a menu system and finishing the demo level to put up on my itch.io soon!
