Video Game: Project DIVE

Game Dev Project

Project DIVE is a action running platformer that entails running, flying, and fighting your way through futuristic landscapes at the fastest speed possible.

This game is still in prototype phase but it emulates the playstation 1 rendering system with shaders in order to give it that retro feel.

This is my current Unity engine solo-dev project that I have been working on for about a year or so and am continuing to put time into.

So far only have one level setup but am working towards having a playable demo coming in the next few months! (on my itch.io page)
